# OwlReads-project
Мобильное приложение для ведения личного читательского дневника с поддержкой сессий чтения, заметок, цитат и интеллектуальных подсказок. Приложение помогает фиксировать чтение, анализировать прогресс и получать персонализированные рекомендации.

## Цель проекта
Создать удобный, надёжный и приватный инструмент для учёта чтения, который сочетает простоту использования для массовых пользователей и продвинутые функции для образовательных и исследовательских сценариев.

## Основные задачи:
* Обеспечить быстрый и минималистичный ввод данных о книгах и сессиях чтения.
* Поддержать офлайн‑работу с последующей синхронизацией и разрешением конфликтов.
* Предоставить удобные инструменты для создания заметок и выделения цитат.
* Реализовать аналитический дашборд и экспорт отчётов.
* Включить интеллектуальные функции (автотегирование, извлечение тем, рекомендации).
